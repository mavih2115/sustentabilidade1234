## 📋 Sobre o Projeto

O **EcoAgro Digital** é uma plataforma web interativa que educa estudantes do Ensino Médio sobre a importância do equilíbrio entre produção agrícola e preservação ambiental. Desenvolvido em **HTML5, CSS3 e JavaScript Vanilla** (sem frameworks), o projeto demonstra conceitos práticos de programação web aplicados a um tema relevante: **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**.

### 🎯 Objetivo

Demonstrar como a tecnologia pode ser usada para:
- Educar sobre sustentabilidade no agronegócio
- Ensinar programação web prática
- Criar interfaces interativas e responsivas
- Engajar estudantes em temas de inovação agrícola

---

## ✨ Características Principais

### 1. 📊 Simulador de Equilíbrio Interativo
Ajuste três variáveis (Água, Tecnologia, Preservação) e veja em tempo real o impacto na:
- Produção agrícola
- Saúde do solo
- Impacto ambiental

### 2. 🚀 Guia de Tecnologias Sustentáveis
Explore 6 tecnologias reais do agronegócio moderno:
- Drones Agrícolas
- Inteligência Artificial
- Sensores IoT
- Agricultura de Precisão
- Energia Renovável
- Rotação de Culturas

### 3. 🎓 Quiz Educativo
Teste seus conhecimentos com 6 perguntas sobre agricultura sustentável com feedback imediato.

### 4. 🎨 Design Moderno
- Paleta de cores futurista (verde-limão, ciano, dourado)
- Animações suaves e transições
- Layout totalmente responsivo
- Acessibilidade garantida

## 🚀 Como Usar

### Opção 1: Abrir Localmente

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/ecoagro-digital.git
cd ecoagro-digital
```

2. Abra o arquivo `index.html` em seu navegador:
```bash
# No Windows
start index.html

# No macOS
open index.html

# No Linux
xdg-open index.html
```

### Opção 2: Usar um Servidor Local

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server

# Acesse: http://localhost:8000
```

### Opção 3: Acessar Online

Visite a versão publicada em:
- GitHub Pages: `https://seu-usuario.github.io/ecoagro-digital`
- Vercel: `https://ecoagro-digital.vercel.app`

## 📖 Como Explorar o Site

### 1. Navegação
Use o menu no topo para navegar entre as seções:
- **Simulador** - Dashboard interativo
- **Tecnologias** - Guia de inovações
- **Quiz** - Teste seus conhecimentos

### 2. Simulador
- Mova os sliders para ajustar Água, Tecnologia e Preservação
- Observe os resultados em tempo real
- Encontre o equilíbrio perfeito!

### 3. Tecnologias
- Clique em "Saiba Mais" para expandir cada tecnologia
- Leia sobre o impacto real de cada inovação
- Clique novamente para colapsar

### 4. Quiz
- Responda as 6 perguntas sobre agricultura sustentável
- Receba feedback imediato (verde = correto, vermelho = incorreto)
- Veja sua pontuação final e mensagem personalizada
- Refaça o quiz quantas vezes quiser
